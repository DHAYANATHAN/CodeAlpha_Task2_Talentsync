# 🚀 TalentSync

## 📌 Overview

TalentSync is a web-based application designed for collaboration and task management. It allows users to manage workflows, interact with data, and perform operations through a simple and user-friendly interface.

---

## ✨ Features

* 👤 User management
* 📋 Task handling
* 🔄 Interactive UI
* 🌐 Web-based system

---

## 🛠️ Tech Stack

* Backend: Java (Spring Boot)
* Frontend: HTML, CSS, JavaScript
* Tools: IntelliJ IDEA, VS Code

---

## 📂 Project Structure

## ▶️ How to Run the Project

### 🔹 Backend

1. Open backend folder in IntelliJ IDEA
2. Run the main class (`@SpringBootApplication`)
3. Server starts at:
   http://localhost:8080

---

### 🔹 Frontend

1. Open frontend folder in VS Code
2. Right-click `index.html`
3. Click **Open with Live Server**
4. Application opens in browser
